# Identité visuelle

Dérivée du mock-up iOS de l'application (voir `presentation/images/simulateur-iphone.png`).

**Logo** : un « diamant-loupe » — le ◆ de la marque in-app devient une loupe (*chercher ses
droits*), blanc sur carré arrondi teal, façon icône iOS.

| Usage | Fichier |
|---|---|
| Icône seule (app, favicon, avatar) | `logo.svg` |
| Logotype horizontal | `logo-texte.svg` |
| Bannière README / réseaux (reprend la barre de recherche du mock-up) | `banniere.svg` |

**Palette** (celle de l'app, `web/style.css`) :

| Rôle | Couleur |
|---|---|
| Teal principal (`--accent`) | `#0b625a` |
| Teal foncé (états actifs) | `#084c46` |
| Teinte claire (`--accent-tint`, pastilles) | `#e1f5ee` |
| Vert badge (`--ok`) | `#0f6e56` |
| Encre / texte | `#1c1c1e` · secondaire `#5c6b67` · atténué `#8a8a8e` |
| Fond « paper » | `#f2f2f7` · cartes `#ffffff` · filets `#dce6e3` |
| Avertissement (démonstrateur) | fond `#fff4e5`, texte `#a24700` |

**Typographie** : pile système (`-apple-system, SF Pro, Helvetica Neue, Arial`), titres en
graisse 700 avec espacement resserré (−1 à −1,5), coins arrondis 12–16 px (34 px pour l'icône à
l'échelle 150 px).
